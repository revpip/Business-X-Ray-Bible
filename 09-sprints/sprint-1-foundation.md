# Sprint 1 — Foundation

## Objective

Create the first real production foundation for the Business X-Ray ecosystem.

## Repositories

- `bx-platform`
- `bx-theme`
- `Business-X-Ray-Bible`

## Platform Deliverables

- Plugin bootstrap
- Autoloader
- Installer
- First database tables
- Admin menu
- Settings framework
- Dashboard template
- Businesses template
- Assessments template
- Admin styling

## Theme Deliverables

- Theme stylesheet header
- Theme bootstrap
- `theme.json`
- Header part
- Footer part
- Index template
- Front page template
- Public styling
- Editor styling

## Bible Deliverables

- Mission
- X-Ray Method
- Product architecture
- UX principles
- Engineering standards
- Sprint plan

## Acceptance Criteria

- Plugin can activate without fatal errors.
- Theme can be recognised by WordPress.
- Admin menu appears.
- Settings page uses registered settings correctly.
- First database tables are created on activation.
- Front page template renders a Business X-Ray hero.
- All sprint decisions are documented.

## Known Limitations

- Assessment submission is not yet implemented.
- Frontend shortcode engine is not yet implemented in the production repo.
- Client portal is not yet implemented.
- AI is not yet implemented.
