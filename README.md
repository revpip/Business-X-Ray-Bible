# Business X-Ray Bible

The source of truth for the Business X-Ray ecosystem.

This repository defines the product philosophy, method, architecture, UX rules, engineering standards, commercial model and sprint plan for Business X-Ray.

## Core Principle

> Business X-Ray helps businesses understand themselves well enough to improve continuously.

## Repositories

- `bx-platform` — WordPress plugin / platform engine
- `bx-theme` — WordPress block theme / public website framework
- `bx-design-system` — design tokens, components and visual language
- `bx-docs` — public and technical documentation
- `bx-ai` — prompts, AI workflows and evaluation
- `bx-demo` — demo site and launch examples
